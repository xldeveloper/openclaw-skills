# Pollclaw Skill for OpenClaw

Agent-first poll orchestration. Create scheduling polls, share links, collect votes, coordinate meetings.

## Installation

```bash
npx clawhub@latest install pollclaw
```

## Examples

```
"Create a poll for team standup next Monday-Friday at 9am or 10am"
"Send the poll to the engineering Slack channel"
"Check the poll results"
"Close the poll and schedule the best time"
```
