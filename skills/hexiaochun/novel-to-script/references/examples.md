# 小说转剧本示例

## 示例 1：战斗场景

### 原文（小说）

> 楚风感受到身后传来的杀气，猛然转身，只见一道黑影已经扑到面前。他来不及多想，本能地侧身闪避，黑影的利爪从他耳边划过，带起一阵腥风。
>
> "好快！"楚风心中一惊，脚下一错，拉开距离。借着月光，他终于看清了来者——那是一头浑身漆黑的妖狼，双眼泛着幽绿的光芒。
>
> "初阶妖兽……还是血化后的。"楚风眯起眼睛，握紧了手中的长剑。

### 转换后（剧本）

```text
1-3 妖兽谷 夜外
人物：楚风、血化妖狼
画面：月光下的山谷，雾气弥漫，气氛诡异

▲近景，楚风背对镜头，突然眉头一皱，眼神一凛
楚风OS：有杀气！

▲快速切换，楚风猛然转身，一道黑影扑面而来
音效：破风声

▲中景，楚风侧身闪避，利爪从耳边划过
音效：利爪破空
特效：爪痕残影

▲近景，楚风脸上闪过一丝惊色
楚风OS：好快！

▲全景，楚风脚下一错，拉开距离，借月光看清来者

▲特写，血化妖狼露出全貌，浑身漆黑，双眼泛着幽绿光芒
音效：低沉的咆哮

▲近景，楚风眯起眼睛，握紧长剑
楚风OS：初阶妖兽……还是血化后的。
```

## 示例 2：对话场景

### 原文（小说）

> "你就是楚风？"龙雨柔高坐在龙椅上，居高临下地看着下方的年轻人，语气中带着几分玩味。
>
> 楚风抬起头，直视着这位传说中的女帝，没有丝毫畏惧。
>
> "正是在下。"
>
> "听说你在边境斩杀了血狼王？"龙雨柔身子微微前倾，眼中闪过一丝兴趣。
>
> "不过是侥幸罢了。"楚风淡淡回应。
>
> 龙雨柔笑了，那笑容中带着几分意味深长："本宫倒觉得，你比你说的要有趣得多。"

### 转换后（剧本）

```text
1-5 帝都皇宫御书房 日内
人物：楚风、龙雨柔、女护卫若干
画面：金碧辉煌的御书房，龙椅高悬，两侧女护卫肃立

▲全景，楚风站在殿中，抬头仰望龙椅上的女帝

▲中景，龙雨柔身穿玄黑鎏金龙袍，高坐龙椅，俯视下方
字幕：女帝 龙雨柔

龙雨柔（玩味）：你就是楚风？

▲近景，楚风抬头直视，目光坦然，毫无畏惧

楚风：正是在下。

▲中景，龙雨柔身子微微前倾，眼中闪过兴趣
龙雨柔：听说你在边境斩杀了血狼王？

▲近景，楚风神色淡然
楚风：不过是侥幸罢了。

▲特写，龙雨柔嘴角微扬，露出意味深长的笑容
龙雨柔（意味深长）：本宫倒觉得，你比你说的要有趣得多。

▲近景，楚风眉头微动，若有所思
```

## 示例 3：系统/金手指场景

### 原文（小说）

> 【叮！检测到宿主吞服聚气丹×3】
> 【修为+90】
> 【当前修为：190/500】
>
> 徐争感受着体内涌动的灵气，嘴角勾起一丝笑意。这个系统，果然是他逆袭的最大依仗。
>
> 【恭喜宿主修为突破100，解锁新技能：瞬步】
> 【瞬步：短距离瞬间移动，冷却时间30秒】
>
> "瞬步？"徐争眼前一亮，"这可是保命的好东西。"

### 转换后（剧本）

```text
1-2 山洞内 夜内
人物：徐争、系统
画面：昏暗的山洞，火把微弱，徐争盘膝而坐

▲中景，徐争手持三颗聚气丹，一口吞下
特效：丹药化作光点融入体内

系统：叮！检测到宿主吞服聚气丹×3

光幕：
修为+90
当前修为：190/500

▲近景，徐争感受体内灵气涌动，嘴角勾起笑意
徐争OS：这个系统，果然是我逆袭的最大依仗。

系统：恭喜宿主修为突破100，解锁新技能：瞬步

光幕介绍：
【瞬步】
效果：短距离瞬间移动
冷却：30秒

▲特写，徐争眼前一亮，露出惊喜神色
徐争：瞬步？这可是保命的好东西。
```

## 示例 4：评论区/弹幕场景

### 原文（小说）

> 视频发布后不到五分钟，评论区就炸了。
>
> "卧槽，这是什么神仙操作？"
> "楼主是开挂了吧？"
> "前排留名，见证大神诞生！"
> "我愿称之为最强！"
>
> 看着不断刷新的评论，林凡嘴角微扬，这波流量，稳了。

### 转换后（剧本）

```text
1-4 林凡公寓 夜内
人物：林凡
画面：现代公寓，电脑屏幕亮着，显示视频发布成功

▲中景，林凡坐在电脑前，看着屏幕

▲特写，手机屏幕，评论数字疯狂跳动
字幕：5分钟后

评论区：网友1：卧槽，这是什么神仙操作？
评论区：网友2：楼主是开挂了吧？
评论区：网友3：前排留名，见证大神诞生！
评论区：网友4：我愿称之为最强！

▲近景，林凡看着不断刷新的评论，嘴角微扬
林凡OS：这波流量，稳了。
```

## 转换要点总结

| 小说元素 | 剧本转换方式 |
|----------|-------------|
| 心理描写 | `角色名OS：内心独白` |
| 环境描写 | `画面：` + `▲` 镜头段落 |
| 动作描写 | `▲` 镜头段落 + 景别 + 动作 |
| 对话 | `角色名（情绪）：台词` |
| 系统提示 | `系统：` + `光幕：` |
| 旁白/叙述 | `旁白：` 或 `角色名VO：` |
| 声音描写 | `音效：` |
| 视觉特效 | `特效：` |
| 时间跳转 | `字幕：三天后` + `【切镜】` |
| 网友评论 | `评论区：网友X：内容` |
