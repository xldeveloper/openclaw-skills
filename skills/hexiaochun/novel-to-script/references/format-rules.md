# 漫剧剧本格式规则（完整参考）

## 1. 文档整体结构

```text
作品名：不写括号补充信息
题材：
类型：
简略梗概：
主要出场人物
  - 主角：
  - 其他角色：
人物简要描述
  - 角色名：简要描述

受众：
情绪承诺（主）：打脸爽/逆袭爽/虐爽/恐惧爽/治愈爽…（只选1）

本集一句话：主角为了【目标】在【规则/限制】下，被【对手/压力】逼到【困境】，最后【变化】并引出【续看问题】
钩子：
增量：
反转/兑现：
续看：
```

## 2. 分场编号规则

### 标准写法
```
1-1 妖兽谷 日外
1-2 温晴住处 日内
```

### 兼容写法
```
1-1 日 外 澜州大石村楚家院子（楚风 楚欣 楚辰）
场景1 林山风景区
```

### 时间标记
- `日外` / `日内` / `夜外` / `夜内`

## 3. 每场必须包含的块

### (A) 场标题行
```
1-2 温晴住处 日内
```

### (B) 人物表
```
人物：角色1、角色2、系统/男声VO（如有）
```

规则：
- 只写本场实际出镜或出声的角色
- 系统、VO、OS 相关"发声体"必须列入人物表
- 群体角色：`女护卫甲/乙/丙`、`官员若干`

### (C) 画面/环境说明（可选但推荐）
```
画面：古香古色的御书房，布置奢华，烛火摇曳
```

### (D) 正文：镜头动作 + 台词

## 4. 镜头/动作描述规则

### 基本格式（`▲` 开头）
```
▲近景，楚风猛然睁开眼，眼中闪过一丝精光
▲全景，众人围成一圈，气氛紧张
```

### 镜头号写法（可选）
```
1，【冬日，漫天飞雪】（字幕：九州大陆，大石村）
2，▲屋檐下，楚风身穿白衣站立
```

### 景别术语
- 远景 / 全景 / 中景 / 近景 / 特写

### 视角术语
- 主观视角 / 旁观视角

### 运动术语
- 推进 / 拉远 / 跟随 / 摇镜 / 切换

### 音效/音乐（独立成行）
```
音效：轰——
BGM：紧张悬疑
环境音：风声呼啸
```

### 特效（独立成行）
```
特效：华光一闪，二人出现
```

## 5. 台词格式规则

### 基础台词
```
温晴：你怎么来了？
楚风（惊讶）：你是谁？
```

### 多人同说
```
楚欣、楚辰（亲昵）：爹爹！
众人（齐声）：是，副阁主！
```

### 旁白/画外音（VO）
```
旁白：三天后，楚风终于抵达了帝都。
楚风VO：这里就是传说中的皇城吗？
```

### 内心独白（OS）
```
楚风：
OS
这家伙果然不简单，我得小心应对。
```

或单行写法：
```
楚风OS：这家伙果然不简单。
```

### 系统提示
```
系统：叮，检测到大量聚气丹，每吃一颗增加30点修为。
```

## 6. 屏幕信息标签

| 标签 | 用途 |
|------|------|
| `光幕：` | 屏幕静态文字 |
| `提示：` | 系统提示/弹窗 |
| `字幕：` | 时间跳切、事件播报 |
| `公告：` | 全服播报/系统广播 |
| `评论区：` | 短视频/弹幕评论 |
| `奖励：` | 奖励清单 |
| `进度：` | 加载/升级进度条 |
| `倒计时：` | 计时显示 |
| `交易内容：` | 交易面板 |
| `聊天大厅：` | 公共频道刷屏 |
| `面板：` | 个人信息/属性面板 |
| `规则：` | 规则条款列表 |
| `排行榜：` | 榜单信息 |

示例：
```
字幕：三天后
提示：是否打开物资箱？
交易内容：矿泉水×2，面包×1
评论区：网友1：帅啊！
评论区：网友2：看起来有点东西啊！
```

## 7. 特殊标记

### 闪回/闪出
```
【闪回】
▲年轻时的楚风站在山顶，意气风发
楚风VO：那时的我，还不知道等待我的是什么。
【闪出】
```

### 转场
```
【切镜】
```

### Q版
```
▲Q版大黄蹦蹦跳跳地走过来
```

## 8. 格式一致性要求

| 项目 | 规则 |
|------|------|
| 镜头标记 | `▲` 与 `△` 二选一，全剧统一 |
| VO 标签 | `VO` 与 `OV` 二选一（推荐 `VO`） |
| OS 标签 | 大小写全剧统一（推荐 `OS`） |
| 镜头号分隔符 | 中文逗号 `，` 或英文逗号 `,` 二选一 |
| 人物列表分隔 | 使用中文顿号 `、` |
| 屏幕字段 | `光幕/光屏` 二选一 |
| 群体角色 | `甲/乙/丙` 或 `1/2/3` 二选一 |
| 角色名 | 同一角色不出现多个写法 |
| 场号 | 同一文档内不可重复 |

## 9. 节奏结构（四段式）

| 段落 | 功能 | 时间建议 |
|------|------|----------|
| 钩子段 | 抓住观众注意力 | 前10秒 |
| 升级段 | 压力递增、信息增量 | 每30-60秒有增量点 |
| 反转/爽点段 | 兑现情绪承诺 | 高潮点 |
| 续看段 | 抛出新悬念 | 结尾 |

## 10. 场数配方

### 2场版
钩子+压力建立 → 兑现爽点+续看钩子

### 3场版
钩子场 → 升级场 → 反转场（兑现）+续看

### 5场版
超短钩子 → 建立/交代 → 行动受挫 → 反转兑现 → 余波/新威胁

## 11. 质量检查标准

### P0（必须修复）
- 缺结构位（缺建立/反应/转场）
- 信息点不清（观众不知道发生了什么变化）
- 锚点不可视化（起/爆/钩不能定格）
- 节奏失约（前10秒不钩、30-60秒无增量、结尾无续看）

### P1（第二优先）
- 人设动机不一致
- 对手压迫方式不稳定
- 道具/能力/地点前后冲突

### P2（最后修复）
- 台词不够口语
- 情绪词堆砌
- 解释过多（应用行动替代）
