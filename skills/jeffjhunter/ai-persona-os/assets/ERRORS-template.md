## Patterns

> Recurring error types and standard fixes

### Permission Errors
- Check file/directory permissions
- Verify user context
- Check ACLs

### API Errors
- Verify credentials
- Check rate limits
- Validate request format

### Network Errors
- Check connectivity
- Verify DNS
- Check firewall/proxy

### Timeout Errors
- Check target availability
- Adjust timeout values
- Check for resource overload
