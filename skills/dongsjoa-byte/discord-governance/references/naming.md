# 命名规范

## 分类命名
```
emoji + 中文名称
```
示例：
- 🔧 编码任务
- 📝 内容创作
- 💰 搞钱大业

## 频道命名
```
小写英文或中文，简短，用连字符分隔
```
示例：
- `#openclaw`
- `#skills`
- `#自媒体运营`
- `#windows-服务器`

**避免**：
- ❌ 过长的名称
- ❌ 特殊字符
- ❌ 全大写

## 子区命名
```
[类型] 简短描述
```

### 类型标签

| 标签 | 用途 |
|------|------|
| `[讨论]` | 一般性讨论 |
| `[任务]` | 具体任务 |
| `[Bug]` | Bug 修复 |
| `[PR]` | PR 相关 |
| `[文章]` | 文章写作 |
| `[调研]` | 调研分析 |

### 示例
- `[文章] skill生态的企业级进化`
- `[PR] 修复heartbeat逻辑`
- `[任务] 小红书测评#003`
- `[讨论] Obsidian同步方案`

## 已完成标记

子区完成后，在名称前加 ✅：
```
✅ [文章] skill生态的企业级进化
```
