# 置顶摘要模板

在每个子区创建时发送并置顶：

```markdown
📌 **【子区名称】**

🎯 **目标**
这个子区要解决什么问题 / 讨论什么话题

📂 **相关资源**
- 文档：[链接]
- 代码：[链接]
- 原讨论：[链接]
- 相关子区：[链接]

📝 **关键结论**
- (讨论过程中持续更新)
- 

🔄 **状态**：进行中 / ✅ 已完成
📅 **最后更新**：2026-02-06
```

## 使用说明

1. **创建时**：填写目标和相关资源
2. **讨论中**：有重要结论时更新「关键结论」
3. **完成时**：
   - 状态改为「✅ 已完成」
   - 补充最终结论
   - 子区名称加 ✅ 前缀

## 示例

```markdown
📌 **[文章] Skill 生态的演进之路**

🎯 **目标**
撰写技术推广文章，介绍 Skill 生态的演进历程

📂 **相关资源**
- 文档：articles/skill-ecosystem.md
- 配图：_assets/skill-ecosystem/

📝 **关键结论**
- 文章完成，11 张配图
- 图片统一风格

🔄 **状态**：✅ 已完成
📅 **最后更新**：2026-02-06
```
