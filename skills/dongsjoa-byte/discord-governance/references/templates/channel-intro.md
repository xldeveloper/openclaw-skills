# 频道简介模板

每个频道的第一条消息（置顶）：

```markdown
# 频道名称

📋 **用途**
这个频道管理什么项目/领域

🔗 **常用链接**
- 仓库：[链接]
- 文档：[链接]
- 相关频道：#xxx

📌 **规范**
- 具体讨论请创建子区
- 子区命名：[类型] 简短描述
- 完成后更新摘要并归档

🗂️ **活跃子区**
(自动归档的不用列，只列当前活跃的)
```

## 示例

```markdown
# my-project

📋 **用途**
项目开发、PR 讨论、问题排查

🔗 **常用链接**
- 仓库：https://github.com/org/project
- 文档：https://docs.example.com

📌 **规范**
- 具体讨论请创建子区
- 子区命名：[类型] 简短描述
- 完成后更新摘要并归档

🗂️ **活跃子区**
- [PR] 修复 heartbeat 逻辑
- [讨论] 新版本更新计划
```
