# 更新摘要流程

## 什么时候更新

- 有重要决定/结论
- 添加了新的相关资源
- 任务状态变化
- 讨论完成

## 步骤

### 1. 找到置顶摘要
子区的第一条置顶消息

### 2. 编辑消息
```
message action=edit \
  channel=discord \
  messageId=<摘要消息ID> \
  message="更新后的完整内容"
```

### 3. 更新内容

**添加结论**：
```markdown
📝 **关键结论**
- 原有结论
- 新增结论 ← 添加这行
```

**更新状态**：
```markdown
🔄 **状态**：✅ 已完成
📅 **最后更新**：2026-02-06
```

**添加资源**：
```markdown
📂 **相关资源**
- 原有链接
- 新文档：[链接] ← 添加这行
```

## 最佳实践

- 结论写简洁，关键信息即可
- 保持格式一致
- 每次更新都改「最后更新」日期
- 完成时记得改状态
