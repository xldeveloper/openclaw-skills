# 归档子区流程

## 什么时候归档

- 任务完成
- 讨论告一段落
- 问题已解决

## 步骤

### 1. 更新置顶摘要
- 状态改为「✅ 已完成」
- 补充最终结论
- 更新「最后更新」日期

### 2. 修改子区名称（可选）
加 ✅ 前缀：
```
✅ [文章] skill生态的企业级进化
```

### 3. 让子区归档
- **自动归档**：不活跃一段时间后自动归档
- **手动归档**：Discord 设置里可以手动归档

## 归档后

- 子区**不会删除**
- 从频道列表隐藏
- 可通过以下方式找到：
  - 搜索
  - 子区列表（频道名右边的 # 图标）
  - 直接链接

## 重新激活

如果需要继续讨论，发一条消息即可重新激活子区。
