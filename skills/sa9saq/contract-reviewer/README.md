# ⚖️ contract-reviewer

Automatically detect risk clauses, summarize terms, and rate legal documents.

## Features

- Flag risky clauses (liability, auto-renewal, data privacy, IP assignment, etc.)
- Plain-English summary of complex legal text
- Risk rating: LOW / MEDIUM / HIGH
- Supports text, PDF, markdown, and web URLs
- Structured output with section references

## Usage

```
"Review this NDA" (attach file)
"Check the ToS at https://example.com/terms"
"Analyze this contract for red flags"
```

## Requirements

- No external dependencies
- File read or web fetch capability for document access
